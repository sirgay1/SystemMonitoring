# SystemMonitoring
Тестовое задание Ядро Команда kvadraOS Desktop (Команда 2)

Приложение для отображения загрузки ресурсов ПС (CPU, память, процессы) в реальном времени.  
Работает на Linux и предоставляет два интерфейса:
- **Веб-интерфейс** – доступен в браузере по адресу `http://localhost:8080`

Программа собирается из исходных кодов на C++ с использованием `make`.

## Требования

- Операционная система: **Linux** (Ubuntu)
- Установленные пакеты: `g++`, `make`

## Установка и запуск

### 1. Установить компилятор и утилиты (если не установлены)

```bash
sudo apt update
sudo apt install make g++
```

### 2. Перейти в папку проекта

```bash
cd /путь/к/проекту/backend
```

### 3. Собрать проект

```bash
make
```

### 4. Запустить приложение

```bash
./sysmon
```
