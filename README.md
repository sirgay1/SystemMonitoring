# SystemMonitoring
Тестовое задание Ядро Команда kvadraOS Desktop (Команда 2)

Приложение для отображения загрузки ресурсов ПС (CPU, память, процессы) в реальном времени.  
Работает на Linux и предоставляет два интерфейса:
- **Веб-интерфейс** (основной) – доступен в браузере по адресу `http://localhost:8080`
- **Консольный интерфейс** (опционально) – аналогичен `top`/`htop`

Программа собирается из исходных кодов на C++ с использованием `make`.

## Требования

- Операционная система: **Linux** (Ubuntu)
- Установленные пакеты: `g++`, `make`, `libncurses-dev` (для консольной версии)
- Подключение к интернету (один раз, для автоматической загрузки библиотек `httplib.h` и `json.hpp`)
